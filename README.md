\# Solver de Catenaria Exacta



Aplicación web para calcular la configuración de equilibrio de un cable con dos tramos y un apoyo intermedio fijo.



\## Características

\- Cálculo exacto de catenaria

\- Visualización de geometría y tensiones

\- Reacciones en apoyos

\- Análisis de precisión



\## Uso

1\. Ingresa los parámetros en la barra lateral

2\. Haz clic en "CALCULAR"

3\. Explora los resultados en las diferentes pestañas
